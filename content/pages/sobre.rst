O que é videos.softwarelivre.org?
---------------------------------

videos.softwarelivre.org_ é um índice de vídeos de palestras em português sobre Software Livre.

Este projeto usa o código do PyVideo.org_ desenvolvido pela comunidade Python.

Onde?
-----

O código do videos.softwarelivre.org_ está hospedado no GitHub.com_.

Quem?
-----

videos.softwarelivre.org_ é mantido por `Paulo Henrique de Lima Santana`_.

Realizei a tradução da interface web e estou inserindo os eventos.

Gostou?
-------

Se gostou dessa iniciativa, você pode fazer uma doação de qualquer valor para mim :-)

Isso ajudará a manter este e outros projetos como a agenda.softwarelivre.org_ e a timeline.softwarelivre.org_

Palestras do FISL
-----------------

As informações das palestras do `FISL - Fórum Internacional Software Livre`_, foram gentilmente cedidas pelo Eduardo Klosowski.
